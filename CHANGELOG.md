# 更新日志

## [1.2.0] - 2025-11-01

### ✨ 新增功能

#### 高级上传系统

实现了三种上传方式，针对不同场景和文件大小：

**1. 智能上传模式选择**
- 文件 < 10MB：自动使用直接上传
- 文件 ≥ 10MB：弹出选择框，可选择：
  - 直接上传（快速）
  - 分片上传（稳定）
  - 断点续传（可恢复）

**2. 分片上传（Chunk Upload）**
- 将大文件分成2MB的小块上传
- 每个分片独立上传，降低内存占用
- 显示分片进度（如：分片 5/10）
- 适合 > 100MB 的大文件

**3. 断点续传（Resumable Upload）**
- 支持上传中断后继续
- 自动检测已上传的分片
- 从断点位置继续上传
- 基于文件哈希识别，即使文件重命名也能续传
- 适合网络不稳定或超大文件（> 500MB）

**技术实现**：
```javascript
// 文件哈希计算（使用SparkMD5）
const fileHash = await calculateFileHash(file);

// 检查断点
const status = await checkUploadStatus(fileHash);
// 跳过已上传的分片
if (uploadedChunks.includes(chunkIndex)) continue;
```

### 📝 技术细节

#### 自定义文件名处理
```python
def secure_filename(filename):
    # 移除危险字符，但保留中文
    filename = re.sub(r'[/\\:*?"<>|]', '', filename)
    # 移除开头的点（防止隐藏文件）
    filename = filename.lstrip('.')
    # 限制长度
    if len(filename) > 255:
        name, ext = os.path.splitext(filename)
        filename = name[:255-len(ext)] + ext
    return filename.strip()
```
